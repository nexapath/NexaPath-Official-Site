# NexaPath Official Site

Vite + React + Tailwind single-page bilingual website for NexaPath (business resource sourcing & matching).

## Dev
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Framework: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
